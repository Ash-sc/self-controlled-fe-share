前言


一切防御机制都是我们为逃避痛苦而向自己撒的谎

唐纳德·梅尔泽（英）

心理防御机制是人类心理中一个普遍而必需的部分，
既给我们提供了保护，帮助我们解决了人类经历中的一些难题，
但同时也阻碍了我们成长以及对满足感的获得。

心理防御机制的介绍

"潜意识"

德国哲学家克里斯托弗·瑞格尔爵士 18世纪 提出潜意识一词。
一百多年前被 西格蒙德·弗洛伊德 当作其精神分析理论的基石。
我们能轻易识破他人的一些潜意识动机，却对自身的动机不明所以。
换言之，我们并不想知道自己潜意识里的内容，如果我们真想知道的话，这些想法和感受根本就不会进入潜意识领域里。
潜意识和意识的冰山模型：潜意识占据主体，意识只是小部分。

"忘记"


忘记这一行为本身泄露了他自始至终就不愿去做那件事
比如：纪念日、日报；虽然有时候确实是因为工作压力大等一些原因
但疏忽和出错往往揭露了一些事实，它们是我们未能意识到，甚至对自己都不愿坦诚的真相

"但是"


当他人以批评的语言/用”总是“、”每件事情“等字眼对我们进行评价时，我们会很明显的发现自己很难做到倾听对方
哪怕对方说的是对的。
“但是” 是一个明显的转折词，它说明：当我们面对痛苦时，极易表现出抗拒心理。

环境 -> 性格 -> 心理防御

“人是这样一类灵长目，拥有漫长而感到无助的婴儿期，和更加漫长而依赖的童年。”
“人类是一种社会性动物”

你或许无法建立长久又满意的关系，不是因为你很难亲近他人，就是缘于他人发现你占有欲太强或“太黏人”。你也可能是那个动不动反应过度却又事后过意不去的人；抑或你封闭了自己，不知道自己真实的感受。可能你一生都在遭受低自尊感，甚至是自我憎恨。

一些常见的心理防御机制

  • 1. 压抑与否认（“认错时的看似态度陈恳”、“吃醋”、“转移话题”、“直接否认”）
    • 人们最常否认的一种感觉是愤怒。
  • 2. 转移与反向（“倾吐”、“buffer”）
    • 举到一个 新妈妈半夜给闹腾的孩子喂奶 的例子
  • 3. 合理化（通过谎言来辩护）
    • 下载一张盗版没什么，毕竟我是个学生党
    • 绩效少是因为领导不喜欢我，而不是因为我不努力
  • 3. 羞耻防御
    • 真遗憾，没法去团建了

如何解除无益的心理防御

  • 心理防御是一生习得的思维习惯
    就像健身和减肥，是一个持久战。
  • 人类不可能一直处于友善状态，愤怒、仇恨、嫉妒与妒忌等同样属于我们情感组成中不可或缺的部分
    比如：我去面试的时候，会比较积极又比较消极。
  • 挑战隐性防御
    比如：在遇到重大的决策时 -> 变得被动 -> 幻想周围围绕着的是一群充满活力能量的人 -> 他们会帮忙解决遇到的一切难题 又或者说，当交给自己的任务在自己不擅长的领域时，会变得精神不集中、拖延、等。
